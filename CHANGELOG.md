# Changelog

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-02

### Added
- Initial release of `country_selector`.
- Support for selecting countries via Page or Sheet widgets.
- Search functionality with fuzzy matching and typo tolerance.
- Built-in localization support.
- Customizable UI elements (styles, search box, flag size).
- Support for favorite countries pinned to the top.
- Platform-aware optimizations (e.g. web flag preloading).
